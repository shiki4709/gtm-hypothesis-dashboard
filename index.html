Step 3: The 3-Question Filter

Q1: Signal in under 1 week?
Can we get a yes/no answer within 7 days?
Q2: Touches ICP directly?
Does it reach AEs at B2B startups — not adjacent personas?
Q3: No new paid tool needed?
Can we run it with free tools or what we already have?
3/3 → Run it
2/3 → Review
0-1/3 → Park it

Step 5: Reading the Signal

UNCLEAR

NO
